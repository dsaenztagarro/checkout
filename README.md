# checkout

Installation
------------

```shell
gem install bundle rake -N
bundle
```

Rake tasks
----------

- `test:integration`: run acceptance tests with `cucumber`
- `test:unit`: run unit tests with `rspec`

Documentation
-------------

1. Run local documentation server: `yard server`

2. [Check documentation](http://localhost:8808)
